# OS v13.0.1 — Responsive UI & Trial Findings Corrective Update

## Corrected in this package
- Adds visible desktop **Record Activity** action while retaining mobile floating `+`.
- Redesigns desktop layouts to use a true dashboard workspace while preserving mobile app flow.
- Makes Dashboard Current Month Cash Flow actual-first with forecast as secondary and adds Actual Net Cash Flow.
- Adds allocation percentages; groups valuation-based assets under Market / Valuation Assets and hides settled receivables from current outstanding display.
- Restores Portfolio Balance History, Group Trends, MoM Growth, and next-month holding estimates.
- Makes Portfolio selected-month total follow Updated Forecast so recorded incidental income/expense is reflected consistently.
- Restores controlled-category Daily & Weekly Spending Rate cards.
- Combines Spending Composition chart and ranked values in one card.
- Adds Expense Control progress bar and budget-deviation colouring in Expense Breakdown.
- Improves Recent Transactions colour/sign display; transfers are shown as internal transfers, not negative portfolio events.
- Replaces inset app icon with new full-bleed `any` and `maskable` icon assets.

No Google Sheets schema changes from OS v13.0.0 are required for this corrective update.
