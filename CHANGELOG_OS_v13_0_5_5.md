# PortOS — OS v13.0.5.5
## Sticky Month Navigation & Mobile Header Compactness

Public-PWA-only enhancement based on PortOS OS v13.0.5.4.

### Sticky month navigation
- Monthly and Portfolio month selectors are integrated into the existing sticky top bar.
- Month navigation remains available while scrolling these pages.
- Dashboard and Settings do not gain a month selector.

### Mobile global header
- The sticky header shows a consistent active-page SVG icon on Dashboard, Monthly, Portfolio, and Settings.
- Hide/Show Values remains available as an eye/eye-off icon.
- Animated Sync and Lock actions remain in the header.
- The floating `+` transaction-entry action remains unchanged.
- Monthly and Portfolio fit page context, month navigation, Hide, Sync, and Lock into one compact sticky row.

### Desktop global header
- Keeps readable text page headings.
- Monthly and Portfolio show month controls in the sticky header beside the page context.
- Existing Record Activity, Hide Values, Sync, and Lock actions remain.

### Deployment
No Apps Script replacement or Google Sheet change is required.
