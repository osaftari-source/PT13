# PortOS — OS v13.0.5.6
## Mobile Header Text Context Refinement

Public-PWA-only refinement based on PortOS OS v13.0.5.5.

### Changed
- Mobile sticky header shows the active page as readable text again (`Dashboard`, `Monthly`, `Portfolio`, `Settings`) rather than a page icon.
- Monthly and Portfolio retain the sticky selected-month navigation in the same global header row.
- Compact sizing is applied where month navigation is displayed so page text and controls fit together on mobile.

### Preserved
- Eye / eye-off Hide Values toggle.
- Animated Sync icon and Sync behaviour.
- Lock icon.
- Floating `+` activity input button.
- Desktop textual header and sticky Monthly/Portfolio month selector.
- All v13.0.5.4 Monthly layout and analysis refinements.

### Deployment
No Apps Script replacement or Google Sheet change is required.
