# OS v13.0.0 Public PWA Files

Upload the files in this folder to the v13 GitHub Pages deployment. This folder contains only the public web app shell, PWA assets, and new app icons. Private configuration and migrated financial data are not included here and load only after authenticated sync.

For mobile installation after deployment:
- Android / Chrome: open the site and choose **Install app** or **Add to Home screen**.
- iPhone / Safari: open the site, tap **Share**, then choose **Add to Home Screen**.
