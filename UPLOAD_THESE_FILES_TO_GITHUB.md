# Portfolio Tracker OS v13.0.3 — Public PWA Upload Files

Upload these public files/folders to your GitHub Pages app location:
- `index.html`
- `app.js`
- `styles.css`
- `manifest.json`
- `service-worker.js`
- `icons/`

Do not upload your private Apps Script, Google Sheets workbook, endpoint, or token to GitHub.

## Update note
The service-worker cache version is `portfolio-tracker-os-v13-0-2`. After upload, open the deployed app and use **Sync**. If the old UI remains cached, close/reopen the installed app or reinstall the PWA shortcut after GitHub Pages finishes publishing.

After publishing, update the private Apps Script backend from the v13.0.3 private package and deploy a new web-app version.
