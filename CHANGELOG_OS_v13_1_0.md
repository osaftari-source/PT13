# PortOS — OS v13.1.0
## Premium Visual Workspace Redesign

Major visual/interface redesign based on the approved concept direction, built on PortOS v13.0.5.6 functionality.

### Visual redesign
- Purpose-built desktop finance workspace with navy shell, persistent sidebar, clean analytical canvas, refined cards and polished action bar.
- Premium mobile finance-app hierarchy with compact cards, dark contextual header, sticky month navigation, bottom navigation and floating activity action.
- Updated navigation icon treatment and PortOS private-use footer on desktop sidebar.

### Dark / System-dark audit
- Corrected Record Activity popup and shared form/dialog modal contrast.
- Applies corrected modal styling to transaction forms, Update Value, balance updates, Receivable forms, confirmation dialogs and connection/PIN workflows.
- Retains readable semantic colours in dark mode: positive green, expense/negative red, action/investment teal-blue and warning gold.

### Functional preservation
- No intentional financial calculation or workflow changes.
- Retains private authenticated backend, month-key integrity, sync behaviour, planning/reallocation logic, month-end readiness, Recent Transactions layout, sticky month navigation and transaction-entry workflow.

### Deployment
Public-PWA-only redesign. No Apps Script replacement or Google Sheet change is required.
